        4. Check the result


#### 3. Create virtual environment

        Steps:
        
        1. Go to root directory 'ecommerce'
        2. Create virtual environment 'venv3922'
        3. Check the result

        5. Checking the content of the 'venv3922'

        Steps:

        1. Go back to root directory 'ecommere'
